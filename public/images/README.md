## 이미지 요구사항

- **형식**: JPG 또는 PNG
- **해상도**: 최소 1920x1080 (Full HD)
- **비율**: 16:9 가로 비율 권장
- **용량**: 500KB - 2MB (웹 최적화)
- **스타일**:
  - 낮 또는 밤 시간대
  - 건물 전경 또는 세부 구조
  - 단청(Dancheong) 색상이 잘 보이는 각도
