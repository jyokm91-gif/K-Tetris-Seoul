## 파일 형식

- **형식**: MP3
- **비트레이트**: 128-192 kbps (용량 최적화)
- **길이**: 2-5분 (루프 재생됨)
- **볼륨**: 정규화된 볼륨 (너무 크거나 작지 않게)
